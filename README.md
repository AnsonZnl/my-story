####    涉及技术点如下
+   vue+vue-cli+element 用于搭建页面
+   axios  用于属于请求
+   vue-router 路由跳转
+   vue-quill-editor  富文本编辑器
+   body-parser  解析请求
+   cookie-parser  cookie相关 express的一个中间件
+   express  node.js应用开发框架
+   mysql  数据库

#### 主要思想
自己用node（express）搭建的服务器，通过路由获取前端页面传过来的参数，再由model层进行处理连接数据库进行操作，返回结果，传给前端，目前实现登录，发表小说，以及个人主页等
